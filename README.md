
# PRO Telegram Stock Bot

بوت احترافي عالي المستوى للسوق الأمريكي.

## المميزات
- تنبيهات اختراق لحظية
- تنبيهات Volume Spike
- تنبيهات Halt
- أخبار الأسهم
- مراقبة Premarket
- أوامر تليجرام احترافية
- Watchlist
- قاعدة بيانات SQLite
- دعم TradingView Webhook
- قابل للتشغيل 24/7

## APIs المطلوبة
- Telegram Bot Token
- Polygon API
- Finnhub API

## التثبيت
```bash
pip install -r requirements.txt
```

## التشغيل
```bash
python main.py
```
